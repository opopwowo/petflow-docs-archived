# 流程圖 Flowcharts & State

> 以流程圖、狀態機與時序圖描述業務邏輯。

## 模組介紹

本模組集中管理業務流程圖、狀態機（State Machine）與時序圖（Sequence Diagram），統一使用 Mermaid 描述，使跨團隊對流程有一致理解。

## 目的

- 繪製核心業務流程圖（BPMN / Flowchart）
- 定義關鍵實體的狀態機（如登記、訂閱、付款）
- 繪製重要互動的時序圖
- 維護流程圖的版本與一致性
