# 流程圖 Flowcharts & State

> 以流程圖、狀態機與時序圖描述業務邏輯。

## 模組介紹

本模組集中管理業務流程圖、狀態機（State Machine）與時序圖（Sequence Diagram），統一使用 Mermaid 描述，使跨團隊對流程有一致理解。

## 目的

- 繪製核心業務流程圖（BPMN / Flowchart）
- 定義關鍵實體的狀態機（如登記、訂閱、付款）
- 繪製重要互動的時序圖
- 維護流程圖的版本與一致性

## 文件清單

- [x] [01_核心業務流程圖](01_核心業務流程圖.md)：租戶開通、寵物入店建檔到售出/交付、官方登記、疫苗提醒與訂閱變更的核心業務流程（FLOW-001～005、JNY-001）。
- [x] [02_狀態機圖](02_狀態機圖.md)：寵物（含軟刪除橫切）、官方登記申請、訂閱、付款與配種的狀態機定義（STM-001～006）。
- [x] [03_時序圖](03_時序圖.md)：登入驗證、下單（售出交易）與通知發送的跨層互動時序（SEQ-001～003）。
- [x] [04_流程圖繪製與命名規範](04_流程圖繪製與命名規範.md)：Mermaid 圖類型選用、節點命名、圖表編號（FLOW/STM/SEQ/JNY）、樣式約定與版本維護規範。
