# Multi-Tenant 多租戶架構

> 定義多租戶資料隔離與安全策略。

## 模組介紹

本模組規範多租戶的隔離模型（共用資料庫、以 tenant_id 隔離為主），確保不同租戶資料絕對隔離、查詢必帶租戶範圍、避免越權存取，是平台安全的基石。

## 目的

- 選定隔離策略（共用 DB / Schema / 行級隔離）
- 強制所有查詢帶 tenant_id 範圍
- 防止跨租戶資料外洩
- 規劃租戶層級設定與客製
