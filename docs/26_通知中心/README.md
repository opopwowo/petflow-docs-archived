# 通知中心 Notification Center

> 統一管理站內、Email 與推播通知。

## 模組介紹

本模組提供統一的通知能力：站內通知、Email、Web Push / 行動推播，支援事件驅動、範本管理、使用者偏好與發送佇列（Cloudflare Queues），服務提醒 / 行銷 / 系統訊息。

## 目的

- 建立統一通知模型與多通道發送
- 以事件驅動觸發通知（如疫苗到期）
- 管理通知範本與多語系
- 支援使用者偏好與退訂
