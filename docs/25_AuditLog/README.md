# Audit Log 稽核日誌

> 記錄所有寫入操作以供追溯。

## 模組介紹

本模組規範稽核日誌：所有建立 / 修改 / 刪除 / 還原操作皆須記錄 who/what/when/where 與變更前後內容，日誌唯讀不可竄改，支援合規查核與資安調查。

## 目的

- 定義稽核事件與必填欄位
- 記錄變更前後（before / after）內容
- 確保日誌唯讀、不可竄改
- 提供查詢、匯出與保留政策
