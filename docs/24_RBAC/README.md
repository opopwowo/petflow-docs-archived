# RBAC 角色權限控管

> 以角色為基礎的存取控制。

## 模組介紹

本模組定義使用者 → 角色 → 權限的授權模型，採最小權限與預設拒絕原則，並與多租戶 / 多店層級結合，控管每個 API 與操作的存取。

## 目的

- 定義角色、權限與其關聯模型
- 建立角色 × 權限矩陣
- 實施最小權限與預設拒絕
- 整合租戶 / 門市範圍的資料層授權
