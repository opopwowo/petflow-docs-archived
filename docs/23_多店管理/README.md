# 多店管理 Multi-Store

> 支援連鎖品牌的多門市管理。

## 模組介紹

本模組支援單一租戶下的組織結構：品牌 / 公司 → 多門市（Store）→ 員工，處理門市層級的資料範圍、庫存 / 寵物歸屬、跨店調度與彙總報表。

## 目的

- 建立組織階層（品牌 / 門市 / 員工）
- 定義門市層級資料範圍與權限
- 支援跨店調度與資源共享
- 提供總部彙總與分店報表
