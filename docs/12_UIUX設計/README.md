# UI/UX 設計 Design System

> 以 Material Design 3 建立一致的設計系統。

## 模組介紹

本模組定義產品的視覺與互動規範，遵循 Material Design 3（Material You），建立 Design Token、元件庫、版面與無障礙標準，確保跨平台一致的使用體驗。

## 目的

- 建立 Material Design 3 設計系統與 Token
- 定義元件庫與互動狀態
- 規範響應式版面（Mobile First）與深色模式
- 確保無障礙（WCAG 2.1 AA）
