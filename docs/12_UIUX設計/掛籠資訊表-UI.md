# 掛籠資訊表 UI 設計 Cage Card UI

> 掛籠資訊表（Cage Card）之畫面與列印樣板設計草案，遵循 [Material Design 3](README.md)。功能規格見 [13 寵物管理 › 掛籠資訊表](../13_寵物管理/掛籠資訊表.md)。

## 1. 設計原則

- **Material Design 3（Material You）**：使用 Design Token(色彩、字級、圓角、間距、Elevation)。
- **Mobile First / 響應式**：手機 / 平板 / 桌機一致;觸控目標 ≥ 48×48dp。
- **深色模式與動態色彩**:支援;列印樣板固定為淺色高對比以利辨識。
- **無障礙**:WCAG 2.1 AA;色彩對比 ≥ 4.5:1,狀態不僅以顏色表示(輔以文字/圖示)。
- **元件狀態完整**:default / hover / focus / pressed / disabled / error。

## 2. 畫面清單

| 畫面 | 說明 |
| --- | --- |
| 卡片列表 | 依門市 / 狀態過濾,顯示縮圖、寵物名、籠位、狀態 chip |
| 卡片編輯 | 左側欄位設定、右側**即時預覽**;含欄位顯示開關 |
| 列印預覽 | 選樣板(A5/A6、直式/橫式)、產生 PDF |
| QR 公開頁 | 消費者掃描後的行動頁(照片、疫苗摘要) |

## 3. 編輯畫面版面（草案）

```text
┌───────────────────────────────────────────────┐
│  App Bar：掛籠資訊表 ‹ 返回        [儲存] [發布]  │
├────────────────────────┬──────────────────────┤
│  設定區（左）            │  即時預覽（右）         │
│  ┌──────────────────┐  │   ┌────────────────┐  │
│  │ 籠位編號  [____]  │  │   │  （卡片樣板預覽） │  │
│  │ 樣板     [A5 直▾] │  │   │  寵物名 / 品種   │  │
│  │ 售價 [開關] [__]  │  │   │  晶片號（鎖定）  │  │
│  │ 顯示欄位（開關群） │  │   │  來源（鎖定）    │  │
│  │  · 售價           │  │   │  QR Code        │  │
│  │  · 疫苗摘要        │  │   └────────────────┘  │
│  └──────────────────┘  │                        │
└────────────────────────┴──────────────────────┘
  手機：設定與預覽改為上下堆疊（Tab 切換）
```

- **法定揭示欄位**(晶片號、來源)在開關群中呈現為**鎖定/停用狀態**,附說明「法規要求,不可隱藏」(對應 API `MANDATORY_FIELD_HIDDEN`)。
- 狀態以 chip 表示:`draft`(灰)、`published`(綠)、`archived`(橙),並附文字標籤。

## 4. 列印樣板

| 樣板 | 尺寸 | 方向 | 建議用途 |
| --- | --- | --- | --- |
| A5 直式 | 148×210mm | 直 | 標準籠位,資訊較多 |
| A5 橫式 | 210×148mm | 橫 | 寬籠位 |
| A6 直式 | 105×148mm | 直 | 小型籠位 |
| A6 橫式 | 148×105mm | 橫 | 小型寬籠位 |

**版面元素(由上而下)**:店家標誌/店名 → 寵物名與品種 → 基本資料(性別、出生、毛色)→ **晶片號碼**與**來源**(法定)→ 疫苗/驅蟲摘要(可選)→ 售價(可選)→ QR Code + 籠位編號。

- 列印為淺色底、黑字高對比;QR Code 最小邊長 ≥ 2cm 以利掃描。
- PDF 於 Cloudflare Workers 產生、暫存 R2、短效簽章連結下載(見 API `POST /cage-cards/{id}/print`)。

## 5. Design Token（沿用系統）

- 色彩:主色 deep purple(見 `mkdocs` 主題與設計系統);狀態色使用語意 Token(success/warning/neutral)。
- 圓角:卡片 12dp;字級遵循 MD3 Type Scale。
- 間距:8dp 基準網格。

> 實際 Token 值以 [12 UIUX 設計](README.md) 之 Design Token 章節為準;本頁僅定義掛籠資訊表的套用方式。
