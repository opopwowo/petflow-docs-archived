# 健康管理 Health Management

> 管理寵物疫苗、病歷與健檢。

## 模組介紹

本模組記錄寵物的疫苗接種、驅蟲、病歷、用藥、健康檢查與提醒排程，支援獸醫與店家追蹤寵物健康狀態，並可觸發到期通知。

## 目的

- 記錄疫苗 / 驅蟲 / 病歷 / 用藥
- 管理健檢與健康指標
- 排程到期提醒（疫苗 / 回診）
- 提供健康時間軸檢視
