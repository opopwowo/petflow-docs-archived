# 測試 Testing & Quality

> 定義測試策略與品質保證。

## 模組介紹

本模組建立測試金字塔（單元 / 整合 / E2E）、測試資料與環境、覆蓋率門檻、效能與安全測試，並將驗收條件對應至自動化測試，確保文件規格被正確實現。

## 目的

- 建立測試策略與測試金字塔
- 定義覆蓋率門檻與品質關卡
- 規劃效能 / 負載 / 安全測試
- 將驗收條件對應到自動化測試
