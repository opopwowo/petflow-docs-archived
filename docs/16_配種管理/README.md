# 配種管理 Breeding Management

> 管理配種、血統與繁殖紀錄。

## 模組介紹

本模組面向專業繁殖者，記錄配種計畫、交配、懷孕、產仔（Litter）、血統譜系（Pedigree）與近親係數，協助合規且健康的繁殖管理。

## 目的

- 記錄配種計畫與交配紀錄
- 管理懷孕 / 產仔（Litter）紀錄
- 建立血統譜系（Pedigree）
- 計算與提示近親繁殖風險
