# 安全性 Security & Compliance

> 定義資安、隱私與法規遵循。

## 模組介紹

本模組統籌平台的安全與合規：身分驗證、授權、加密、機密管理、輸入驗證、OWASP 防護、個資保護（含寵物登記相關規範）與事件應變，是所有模組的橫切要求。

## 目的

- 定義驗證 / 授權 / 加密標準
- 落實機密管理與輸入驗證
- 對照 OWASP Top 10 防護
- 規劃個資保護與資安事件應變
