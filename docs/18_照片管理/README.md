# 照片管理 Photo Management

> 管理寵物影像的上傳、儲存與處理。

## 模組介紹

本模組負責影像的上傳、儲存（Cloudflare R2）、縮圖與壓縮、存取授權與生命週期管理，供寵物檔案、健康、登記等模組使用。

## 目的

- 設計影像上傳與驗證流程
- 以 R2 儲存並產生縮圖 / 多尺寸
- 控管影像存取權限（簽章 URL）
- 管理影像生命週期與容量
