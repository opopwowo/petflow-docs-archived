# 市場分析 Market Analysis

> 分析目標市場規模、競爭格局與產業趨勢。

## 模組介紹

本模組評估寵物科技（PetTech）與寵物事業管理 SaaS 的市場機會，界定目標客群、競品優劣勢與切入策略，作為商業模式與產品優先序的依據。

## 目的

- 量化市場規模（TAM / SAM / SOM）
- 盤點主要競品並進行優劣勢分析
- 歸納產業趨勢與法規環境（寵物登記、個資）
- 定義目標市場區隔與進入策略

- [x] [01_市場規模TAM_SAM_SOM試算](01_市場規模TAM_SAM_SOM試算.md)：以雙軌估算法量化 TAM/SAM/SOM 與三年 ARR 目標，並登錄待驗證假設。
- [x] [02_競品分析表](02_競品分析表.md)：盤點通用 POS/CRM、國際寵物軟體與 Excel/LINE 手工管理三類競爭者，導出差異化主軸。
- [x] [03_SWOT分析](03_SWOT分析.md)：優劣勢與機會威脅盤點，以 TOWS 交叉矩陣轉化為可執行策略與風險應對。
- [x] [04_產業趨勢與法規環境調查](04_產業趨勢與法規環境調查.md)：寵物產業結構性趨勢與台日東南亞法規環境，界定「合規即賣點」機會。
