# 付款系統 Payment System

> 處理金流、發票與對帳。

## 模組介紹

本模組整合金流服務商，處理付款、退款、發票開立與對帳，確保交易安全（不自存卡號）、可追溯與符合稅務規範。

## 目的

- 整合金流（信用卡 / 行動支付）
- 處理付款 / 退款 / 失敗重試
- 開立電子發票與收據
- 提供對帳與財務報表
