# 寵物管理 Pet Management

> 管理寵物基本檔案的核心模組。

## 模組介紹

本模組維護寵物的完整檔案：品種、性別、出生日期、晶片號、體重、照片、所屬飼主與狀態。它是健康、配種、登記、照片等模組的核心實體（Aggregate Root: Pet）。

## 目的

- 定義寵物資料模型與生命週期狀態
- 支援寵物建立 / 查詢 / 更新 / 軟刪除
- 關聯飼主、健康、配種、登記、照片
- 提供搜尋、篩選與標籤管理
