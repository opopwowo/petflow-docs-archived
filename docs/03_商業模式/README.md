# 商業模式 Business Model

> 定義收費模式、訂閱方案與單位經濟。

## 模組介紹

本模組以 Business Model Canvas 描述 PetFlow 如何創造、傳遞與獲取價值，聚焦 B2B SaaS 訂閱制的方案分層、定價與獲利結構。

## 目的

- 建立商業模式畫布（Business Model Canvas）
- 設計訂閱方案分層（Free / Pro / Enterprise）
- 定義單位經濟（CAC、LTV、毛利、Churn）
- 規劃加值服務與收入來源
