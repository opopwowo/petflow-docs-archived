# 商業模式 Business Model

> 定義收費模式、訂閱方案與單位經濟。

## 模組介紹

本模組以 Business Model Canvas 描述 PetFlow 如何創造、傳遞與獲取價值，聚焦 B2B SaaS 訂閱制的方案分層、定價與獲利結構。

## 目的

- 建立商業模式畫布（Business Model Canvas）
- 設計訂閱方案分層（Free / Pro / Enterprise）
- 定義單位經濟（CAC、LTV、毛利、Churn）
- 規劃加值服務與收入來源

## 文件清單

- [01_商業模式畫布](01_商業模式畫布.md)：以 Business Model Canvas 九宮格描述 PetFlow 的價值主張、客群、通路與獲利結構。
- [02_訂閱方案與定價策略](02_訂閱方案與定價策略.md)：Free / Starter / Pro / Enterprise 四層方案定價（SSOT）、年繳 83 折與升降級規則。
- [03_單位經濟模型](03_單位經濟模型.md)：CAC、LTV、Churn、NRR 等指標口徑、基準估算與敏感度分析。
- [04_收入來源與加值服務清單](04_收入來源與加值服務清單.md)：訂閱收入與 AI 加購、簡訊額度、儲存擴充、登記代辦四項加值服務明細。
- [05_成本結構分析](05_成本結構分析.md)：COGS 與 OPEX 拆解、毛利率估算與損益兩平推導。
