# 使用者角色 Personas & Roles

> 定義使用者 Persona 與系統角色。

## 模組介紹

本模組描述產品的目標使用者畫像（Persona）與系統內的角色（Role），是 RBAC 權限設計與 UX 流程的基礎。涵蓋平台管理者、店主、店員、獸醫、繁殖者、飼主等。

## 目的

- 建立關鍵 Persona（目標、痛點、情境）
- 定義系統角色與職責邊界
- 對應角色與功能權限（為 RBAC 鋪路）
- 作為 User Story 與 UX 的角色來源
