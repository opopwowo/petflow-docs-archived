# 飼主管理 Owner Management

> 管理飼主 / 客戶資料與關係。

## 模組介紹

本模組維護飼主（客戶）的聯絡資訊、與寵物的關係、消費與互動紀錄，並嚴格遵循個資保護規範。是 CRM 與付款、訂閱模組的基礎實體。

## 目的

- 定義飼主資料模型與聯絡資訊
- 維護飼主與寵物的多對多關係
- 記錄互動 / 消費歷史（CRM）
- 落實個資保護與同意管理
